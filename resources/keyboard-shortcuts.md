# ⌨️ Keyboard Shortcuts

## OS & Browser
| Action | macOS | Windows/Linux |
|---|---|---|
| Copy | ⌘+C | Ctrl+C |
| Paste | ⌘+V | Ctrl+V |
| Undo | ⌘+Z | Ctrl+Z |
| Refresh | ⌘+R | Ctrl+R / F5 |
| Force Refresh | ⇧⌘R | Ctrl+Shift+R |
| New Tab | ⌘+T | Ctrl+T |
| Close Tab | ⌘+W | Ctrl+W |

## VS Code
| Action | macOS | Windows/Linux |
|---|---|---|
| Command Palette | ⇧⌘P | Ctrl+Shift+P |
| Open File | ⌘P | Ctrl+P |
| Toggle Terminal | Ctrl+` | Ctrl+` |
| Format Document | ⇧⌥F | Shift+Alt+F |
| Comment Line | ⌘/ | Ctrl+/ |
